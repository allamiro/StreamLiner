Get started in 60 seconds

Drop a single binary or run the Docker image. Define a tiny flow with input → parser → output. Ship normalized events to your SIEM.

1. Run

Docker: docker run -p 514:514/udp ghcr.io/allamiro/streamliner:latest

2. Configure

Create streamliner.ini with your input, parser, and output blocks.

3. Ship

Point your agents to StreamLiner; verify events in your SIEM index.

Why StreamLiner?

Deterministic parsing

Strict schema mapping (CEE/ECS/CLS) to keep dashboards consistent across tenants.

Zero vendor lock-in

Swap outputs (OpenSearch/Elastic/S3/Parquet) without changing your pipelines.

Tiny footprint

Single binary or container. No JVM. Memory-safe and ops-friendly.

Enrich on the edge

GeoIP, ASN, Threat‑intel tagging, and dedup built-in.
